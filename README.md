# 👗 FCS - AI 기반 의류 추천 쇼핑몰

> AI가 사용자의 패션 사진을 분석해 유사한 스타일의 옷을 추천해주는 웹사이트

<br>

## 📋 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 개발 기간 | 2023.03 ~ 2023.06 |
| 팀 구성 | 3인 팀프로젝트 |
| 담당 역할 | 기획/전체 조율, 프론트엔드 개발 |

<br>

## 📌 프로젝트 소개

팀원 모두 패션에 관심이 많아, AI와 웹 개발 역량을 결합하여
사용자가 자신의 사진을 업로드하면 AI가 유사한 스타일의 의류를 추천해주는 쇼핑몰을 개발했습니다.

<br>

| 분야 | 기술 |
|------|------|
| Frontend | HTML, CSS, JavaScript |

## 👤 내가 맡은 역할

### 기획 및 전체 조율
- 프로젝트 아이디어 기획 및 방향 설정
- 팀원 간 역할 분배 (Web / AI 파트) 및 일정 관리
- 회의 진행 및 의견 조율

### 프론트엔드 개발
- 메인 페이지 레이아웃 설계 및 구현
- 상의/하의 카테고리 분류 페이지 구현
- 상품 이미지 수집 및 최적화 (Kaggle, 오픈소스 활용)
- JavaScript로 카테고리 토글 기능 구현 (`classList` 활용)
- 오늘의 상품 추천 섹션 구현
- `position: sticky` + `z-index`를 활용한 스크롤 고정 헤더 구현

<br>

## ✨ 주요 기능

- **카테고리 분류**: 상의(맨투맨, 후드티, 반팔티) / 하의(청바지, 슬랙스, 츄리닝, 반바지)
- **카테고리 토글**: 버튼 클릭 시 카테고리 펼침/접힘 + 화살표 애니메이션
- **현재 위치 표시**: 선택한 카테고리를 색상으로 구분하여 UX 향상
- **오늘의 상품 추천**: 메인 페이지에서 다양한 상품을 한눈에 확인
- **AI 의류 추천** *(개발 중)*: 사용자가 사진 업로드 시 유사한 스타일의 옷 추천

<br>

## 🤖 AI 모델 개요 *(팀원 담당)*

- **MediaPipe**로 사람의 자세 landmark 추출 → 의류 영역(ROI) 특정
- **Canny Filter**로 의류 윤곽선 추출
- **MobileNet + ImageNet 전이학습**으로 의류 분류 모델 구현
- **코사인 유사도**로 가장 유사한 의류 이미지 추천

<br>
